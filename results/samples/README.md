# Generated X-Ray Samples

This directory contains sample images generated by the GANs.

## Contents

- DCGAN generated images (64×64)
- WGAN-GP generated images (256×256)
- Comparison grids

## Sample Images

After training, this directory will contain:

```
samples/
├── dcgan/
│   ├── epoch_10.png
│   ├── epoch_20.png
│   ├── epoch_30.png
│   ├── epoch_40.png
│   └── epoch_50.png
├── wgan/
│   ├── epoch_50.png
│   ├── epoch_100.png
│   ├── epoch_150.png
│   └── epoch_200.png
└── comparison/
    └── dcgan_vs_wgan.png
```

## Generating New Samples

```bash
# Generate DCGAN samples
python src/dcgan/generate.py \
    --weights models/dcgan_generator.pth \
    --num_images 100 \
    --output results/samples/dcgan/

# Generate WGAN samples
python src/wgan/generate.py \
    --weights models/wgan_generator.pth \
    --num_images 100 \
    --output results/samples/wgan/
```

## Visual Quality Progression

The quality of generated images improves over training epochs:

| Epoch | DCGAN Quality | WGAN-GP Quality |
|-------|--------------|-----------------|
| 10 | Noisy, basic shapes | Noisy |
| 50 | Recognizable structures | Basic lung shapes |
| 100 | - | Good structure |
| 200 | - | High quality, realistic |

## Note

Sample images are stored using Git LFS or can be regenerated using the provided scripts.
