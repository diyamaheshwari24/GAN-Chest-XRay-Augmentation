## Image Guidelines

- Use high-resolution PNG format (300 DPI for print, 72 DPI for web)
- Keep file sizes under 1MB each
- Use consistent color schemes
- Include figure captions in documentation

## Creating Architecture Diagrams

Recommended tools:
- **draw.io** - Free, web-based
- **Lucidchart** - Professional diagrams
- **Matplotlib** - Programmatic generation
- **PowerPoint/Keynote** - Quick sketches
